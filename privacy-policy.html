Política de Privacidade

Data da última atualização: 08 de maio de 2025

Esta Política de Privacidade descreve como o aplicativo TopTechura Extrator coleta, utiliza e protege as informações dos usuários ao utilizar nossos serviços integrados ao Facebook/Meta.

1. Coleta de Informações

O aplicativo pode acessar determinados dados fornecidos pelos usuários por meio das permissões autorizadas pelo Facebook, tais como: nome, e-mail, ID do usuário, lista de eventos e outras informações relacionadas.

Esses dados são acessados exclusivamente com o consentimento do usuário e são utilizados unicamente para fins funcionais, conforme proposto pelo aplicativo.

2. Uso das Informações

As informações coletadas são utilizadas apenas para:

  • Fornecer e melhorar os serviços oferecidos pelo aplicativo;
  • Personalizar a experiência do usuário;
  • Garantir o funcionamento adequado das funcionalidades integradas ao Facebook.

3. Compartilhamento de Dados

Não compartilhamos, vendemos ou transferimos suas informações pessoais a terceiros. Os dados são utilizados exclusivamente no contexto funcional do aplicativo, sem fins comerciais.

4. Armazenamento e Segurança

Este aplicativo não armazena dados pessoais em servidores próprios. Toda manipulação de dados ocorre de forma temporária e segura, conforme os padrões recomendados pelas plataformas Meta/Facebook.

5. Consentimento

Ao utilizar o aplicativo, o usuário concorda com os termos desta Política de Privacidade e autoriza o acesso às permissões necessárias para seu funcionamento.

6. Direitos do Usuário

O usuário pode, a qualquer momento, revogar permissões concedidas diretamente nas configurações de privacidade do Facebook, ou desinstalar o aplicativo caso não concorde com esta política.

7. Alterações nesta Política

Esta política poderá ser atualizada periodicamente. Recomendamos que os usuários consultem esta página para se manterem informados sobre qualquer alteração.
